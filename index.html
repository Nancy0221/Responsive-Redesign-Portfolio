Overview

In this assignment, a responsive redesign was carried out on the original webpage, which improved the problems of low contrast, unclear text, repeated buttons, unclear partitions and adaptability to various screen sizes.


Purpose

The purpose of this assignment is to make the text on the page clearer and readable through improvements, allowing users to go to the desired section through the navigation bar, and allowing the web page to be displayed well on any screen size.


About the Web Page

  1. Introduction: This is a food ordering web page.
  2. Write a sentence about why you chose this website: I chose this website because it is a food ordering website and many people may use it, so improvements to it may enhance the experience for lots of users.

Finding Problems

  1. On the home page of the website, the contrast between image and text is low.
  2. The location of some buttons on this website is redundant. For example, user can go to menu by clicking the picture of the food, but there also has a button below the picture.
  3. This home page can be set a side bar at the top of so that users can go to different parts in the same page by clicking the button on the side bar.
  4. Didn't divide different sections appropriately.

Usability & Accessibility

WebAIM WAVE's diagnosis shows that the contrast between the pictures and text on the homepage of the website is low. Secondly, when the menu is displayed, the contrast between the yellow font and the black background is also low. Also, there are very few places where the first level heading and form tags are missing, and there are even a few empty links on the bottom images. I agree with these diagnoses because I also noticed these issues when I first viewed the page. In addition, the lack of labels may cause reading difficulties for blind readers, such as the inability to recognize main headings and tables. Finally, I think that the images without appropriate link can be deleted to prevent users from misunderstanding the use of these images.

Step 6:

Separate the text and background picture so the user can see both of them clearly. In addition, I added responsive design for the new webpage. The upper one is for large screen (desktop and tablet), and the bottom one is for small screen (phone).

Interesting experience: The original design may not allow readers to see the ingredients clearly, but they can clearly see the price at the first glance. Readers will be more willing to pay for the burger if they can be convinced that the burger is worth the price by first reading the ingredients.
